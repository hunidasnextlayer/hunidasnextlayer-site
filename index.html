Competências Técnicas

Sistemas Operativos e Infraestruturas

  • Administração de sistemas Microsoft (Windows Server) e Linux
  • Domínio de Active Directory e do ecossistema Windows Server (DNS, GPO, DHCP, WSUS, etc.)
  • Gestão de redes informáticas e desenho de arquiteturas de rede
  • Routing e switching, com conhecimento sólido de redes LAN/WAN
  • Configuração de proxy reverso com Nginx Proxy Manager
  • Alojamento e gestão de websites com CloudPanel

Virtualização, Backups e Alta Disponibilidade

  • Gestão e implementação de ambientes virtualizados, Hyper-V, vCenter
  • Implementação e manutenção de clusters de alta disponibilidade
  • Estratégias de backup e recuperação com Veeam Backup & Replication

Ferramentas Open Source

  • GLPI para inventário e gestão de ativos informáticos
  • Zabbix para monitorização de sistemas e dispositivos
  • Apache Guacamole para acesso remoto seguro sem VPN
  • Wazuh para segurança, deteção e resposta a ameaças
  • Implementação de soluções personalizadas com ferramentas open-source
